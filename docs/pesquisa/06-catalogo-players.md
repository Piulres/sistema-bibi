## Concorrentes mais próximos do modelo Bibi

1. **ERPMed / Centtralmed** — Pay Per Use + portais (referência de negócio)
2. **Conexa** — B2B corporativo + tele + dados (sem Pay Per Use nativo)
3. **Vitta / Pipo Saúde** — benefícios corporativos + portal empresa

Nenhum combina publicamente **Pay Per Use + precificação por empresa + 4 portais + white label multi-tenant** na mesma proposta (ver [`01-matriz-competitiva.md`](01-matriz-competitiva.md)).
